# Cheap-Yellow-MP3-Player
Cheap Yellow Display (ESP32-2432S028R) used as an MP3 player.

Coded by Claude AI.

It takes about 20 seconds to connect to a Bluetooth speaker in pairing mode, please be patient. 

Ctrl+F in the .ino "Speaker name" and replace "Your speaker name here" with your speaker name there, using exact capitalization, same name as you'd see if you connected to your phone.

ShuffleCYD v2 — MP3 Player for ESP32-2432S028 (CYD)

Features: BT audio, spectrum visualizer, progress bar, playlist browser, near-gapless playback

Libraries: ESP8266Audio, ESP32-A2DP, TFT_eSPI, XPT2046

Partition: "Huge APP (3MB No OTA)"
